# **NotesGym**
> ## *Simple and useful application for taking gym notes*
### **NotesGym Purpose**
The purpose of this application is to allow its users to enter gym notes about the number of repetitions, sets and weight in a simple way, so they can consult it the next time they go to the gym. The application displays statistics and graphs of your progress.
